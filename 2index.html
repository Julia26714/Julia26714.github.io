I learned alot from this class and had alot of fun while doing it, I learned about HTML which stands for Hyper Text Markup language and HTML has 5 basic parts that are necessary. These being the Head, Title, Body, Meta, and ofc HTML. The Head contains all information that won't appear on the screen. The Body contains all the content available to readers. The Meta contains all information such as character set, page description, keywords and more. The acronym HTML stands for "Hypertext Markup language". Css is a coding language primarily used to style and layout web pages. While working with HTML on VS Code you can use the tag "style" to tap into the CSS function and use CSS functions to go with your HTML file the acronym CSS stands for "Cascading Style Sheets"
